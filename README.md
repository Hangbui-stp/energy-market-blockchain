# Energy Market Blockchain System Green

A full-stack blockchain-based energy trading system deployed on **Ethereum Sepolia testnet**, integrating **smart contracts**, **backend services**, **frontend applications**, and **PostgreSQL database**.

## 1. System Overview

This project simulates an **energy marketplace** where users can:
- Mint and trade energy NFTs
- Participate in an energy marketplace
- Join a loyalty program
- Access a lending/credit system via a bank portal
- Store immutable transaction data on blockchain

The system consists of:
- **Smart Contracts** (Solidity, Foundry)
- **Backend** (Spring Boot, Web3j)
- **Frontend Marketplace** (Next.js)
- **Bank Portal** (Next.js)
- **Database** (PostgreSQL)
